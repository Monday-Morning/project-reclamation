# Permissions

## User Section

| Permission Name   | Description                     |
| ----------------- | ------------------------------- |
| user.read.private | Can read non-public users' data |
| user.read.admin   | Can read users' admin fields    |
| user.write.self   | Can update own account          |
| user.write.all    | Can update all user data        |

---
